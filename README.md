# Unveiling the Bright Spot with Mie Scattering

This repository contains the Python notebooks used to analyze the relation between Mie Scattering and the Arago/Poisson Spot, in here presented with the 2024 IPT name, Bright Spot. The notebooks demonstrate simulations and visualizations related to light propagation and scattering.  

## Notebooks

1. **[Introduc.ipynb](./Introduc.ipynb)**  
   This notebook presents simulations of light propagation over a 1D object using Fourier optics. Images and visualizations of the light field are included.

2. **[MieFigure.ipynb](./MieFIgure.ipynb)**  
   This notebook demonstrates how to calculate light propagation using **Mie scattering** theory and show all calculations using x polarized incident plane wave.
   
3. **[MieFigure - General.ipynb](./MieFigure%20-%20General.ipynb)**  
   This notebook demonstrates how to calculate light propagation using **Mie scattering** theory and show all calculations using y and circular polarized incident plane wave.

3. **[ComparaCirc-Esf.ipynb](./ComparaCirc-Esf.ipynb)**  
   This notebook contains comparisons between the well-known 1D bright spot model and 3D propagation using Mie scattering.
